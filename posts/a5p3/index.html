Demo Scenario Bill of Materials

We propose using the following components for our project demo scenario:

  • Bluetooth Headphone for Voice Interface:
    This can be purchased from here, perhaps, or here.
  • Touch/Pressure Sensor:
    Positioned on the handle to act as an emergency stop button when the user firmly press it. It can be replaced by a pushbutton.
    We could purchase this one, for example. We would also need an Arduino to handle button presses.
  • Handle:
    We will also need to 3d-print or buy a guiding handle to attach to the robot's wrist.
    Price: between $5 and $10.
