Manual Grasping Using Our Interface

Our videos show:

  1. The use of our manual grasping interface to pick up a simulated object in RViz.
  2. The use of our manual grasping interface to pick up a real object (with camera focus on RViz).
  3. The use of our manual grasping interface to pick up a real object (with camera focus on the real robot).

Key features of note:

  • The pose reachability is indicated by the gripper color (green when reachable, red when not).
  • The gripper interface was sufficient to successfully pick up a real object.
